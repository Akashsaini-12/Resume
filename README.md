## üìã Installation & Usage

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd resume
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm start
   ```

4. **Generate PDF**
   - Click the "Download Resume" button in the top-right corner
   - The resume will be generated as a PDF file
